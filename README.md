# NewsApp
Simple app to show news and bookmark them to read later.

## Technologies
- MVI architecture (Bidirectional)
- Navigation
- ViewModel
- Flow (StateFlow)
- Room
- Retrofit & OkHttp
- Hilt & Dagger
- LeakCanary
